About This Web App

Home About Count

This web application is built using Flask and Redis, and is containerized with Docker. It demonstrates real-time visitor counting using Redis as a persistent storage solution.

I am learning how to build scalable and portable web applications using containers, and how to separate configuration from code to make apps more flexible.

The counter page shows how Redis can track data dynamically, while the home page provides a welcoming and aesthetic experience.

By using Docker Compose, we can run multiple services together, making this app easy to scale and maintain across different environments.

My goal with this project is to improve my DevOps skills while creating a visually pleasing and functional web application.
